# Shagun Chem — Landing (Next.js + Tailwind)

Minimal, responsive single-page landing built with Next.js (App Router), Tailwind CSS, Framer Motion, and Lucide React.

Getting started

1. Install dependencies:

```bash
npm install
```

2. Run dev server:

```bash
npm run dev
```

Notes
- Colors and fonts are configured in `tailwind.config.js` and `styles/globals.css`.
- Components live in `/components` and the main page is `/app/page.jsx`.
